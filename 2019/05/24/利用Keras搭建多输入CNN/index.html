利用Keras搭建多输入CNN

更新于 2020-02-29 | 分类于 Neural Networks | 评论数：

Keras是一种比较高阶的封装，是一种深度学习框架，其可以架构在Tensorflow, Theano等多种框架上，便于快速的上手和开发。

在这里，利用Keras动手搭建一个CNN模型，Keras模型可以通过两种方式搭建，一是直接采用keras.model.sequential搭建模型，这样简单清晰；二是利用函数式API搭建，函数式API主要用于搭建复杂模型，具有更高的灵活性。

在我做的实验中，是要实现一个关系抽取的模型，有两个输入，一个主输入，一个辅输入，因此使用函数式API进行编程。

首先，输入的张量形式为(batchsize, None, 300)，这里None表示大小不定，是一个变量。在我做的关系抽取实验中，None表示变长的句子长度。

经过卷积操作，卷积核个数为200，张量的形式变为(batchsize, None - 2, 200)，由于卷积操作的进行，句子的长度变为None - 3 + 1，而每一个卷积核生成了一个特征向量，因此共200个。

然后对于每一个卷积核生成的向量，进行全局最大池化，这样，得到的张量就不再与句子长度有关。

每一层的张量形式均得到清晰地描述，而且看到共需训练的参数为207,919个。

这个CNN模型是对Relation Classification via Convolutional Deep Neural Network的一种实现，利用这个模型，可以实现关系分类，进行一些关系抽取任务。
